# 🛍️ Retail Sales SQL Analysis

## 📌 Project Overview

This project demonstrates an end-to-end SQL analysis of a Retail Sales dataset using PostgreSQL.

The objective of this project is to perform data validation, data cleaning, and business analysis using SQL to gain meaningful insights into sales performance, customer behavior, and product trends.

The project follows a complete analytics workflow—from importing raw data into PostgreSQL to answering real-world business questions through SQL queries.

---

## 🎯 Objectives

- Import and manage data using PostgreSQL.
- Validate and clean raw sales data.
- Identify and handle missing values.
- Detect duplicate and invalid records.
- Analyze sales performance using SQL.
- Generate business insights from transactional data.

---

## 📂 Dataset Information

The dataset contains retail sales transactions with the following information:

- Transaction ID
- Sale Date
- Sale Time
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price Per Unit
- Cost of Goods Sold (COGS)
- Total Sale


---


## 🔄 Project Workflow

1. Created the database and imported the dataset.
2. Validated the data by checking:
   - Missing values
   - Duplicate records
   - Invalid values
   - Date range
3. Cleaned the dataset by removing invalid records and correcting column names.
4. Performed business analysis using SQL.
5. Generated business insights from query results.

---

## 📊 Business Questions

1. How much total revenue has the company generated?
2. Which product category generates the highest revenue?
3. Which customer age group spends the most?
4. Which month generated the highest revenue?
5. Who are the Top 10 customers based on total spending?
6. What is the average order value for each category?
7. Which product category is most popular among male and female customers?
8. What are the Top 5 sales dates based on total revenue?
9. Which product category sold the highest quantity of products?
10. Which customer made the highest number of purchases?

---

## 💡 Key Insights

- The dataset generated a total revenue of **₹911,720** from **1,997 transactions**, with an average order value of approximately **₹456**.
- **Electronics** generated the highest revenue among all product categories.
- Customers aged **46–60** contributed the highest overall spending.
- **December** recorded the highest monthly revenue during the analysis period.
- **Clothing** was the most frequently purchased product category.
- The **Top 10 customers** contributed significantly to the overall sales revenue.
- Customer **ID 3** recorded the highest number of purchases, indicating strong customer engagement.
